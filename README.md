# Superstore Sales Dashboard 📊

## Project Description
An interactive Excel dashboard designed to analyze over 9,000 retail transactions. This tool helps in tracking Key Performance Indicators (KPIs) like Total Sales, Profit, and Profit Margins.

## Features
- **Dynamic Slicers:** Filter by Region, Category, and Segment.
- **Trend Analysis:** Monthly sales performance visualization.
- **Profitability Insights:** Deep dive into top-performing sub-categories.
